# CookOverflow

Простой учебный сайт на Laravel в формате "вопрос - ответ" для поваров.

На сайте можно:

- смотреть список вопросов;
- искать вопросы и ответы;
- фильтровать вопросы по тегам;
- регистрироваться и входить в аккаунт;
- задавать вопросы;
- добавлять ответы;
- ставить лайки и дизлайки;
- смотреть профиль пользователя.

## Что нужно для запуска

- PHP
- Composer
- SQLite или MySQL

Проект сейчас настроен на `SQLite`.

## Первый запуск

Открой терминал в папке проекта:

```powershell
cd C:\cursach
```

Установи зависимости:

```powershell
composer install
```

Если файла базы данных еще нет, создай его:

```powershell
New-Item database\database.sqlite -ItemType File
```

Сгенерируй ключ приложения:

```powershell
php artisan key:generate
```

Выполни миграции:

```powershell
php artisan migrate
```

Если нужны тестовые данные:

```powershell
php artisan db:seed
```

## Запуск проекта (кратко и по делу)

1) Подготовь базу (если запускаешь впервые):

```powershell
php artisan migrate
php artisan db:seed
```

2) Запусти сервер (рекомендуется для этого проекта):

```powershell
php -S 0.0.0.0:8088 -t public
```

3) Открой сайт в браузере:

```text
http://127.0.0.1:8088
```

Важно:

- `http://0.0.0.0:8088` в браузере не открывается (это нормальное поведение);
- `0.0.0.0` используется только как адрес прослушивания сервера;
- в браузере используй `127.0.0.1` или `localhost`.

## Альтернатива через Artisan

```powershell
php artisan serve --host=127.0.0.1 --port=8088
```

Если порт занят, поменяй его, например на `8090`, и открой:

```text
http://127.0.0.1:8090
```

## Основные пути

- главная страница: `/`
- регистрация: `/register`
- вход: `/login`
- создание вопроса: `/questions/create`
- профиль пользователя: `/users/{id}`

## Если нужен MySQL

По умолчанию проект использует `SQLite`.

Если хочешь переключить его на `MySQL`, открой `.env` и измени:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=имя_базы
DB_USERNAME=root
DB_PASSWORD=
```

Потом выполни:

```powershell
php artisan migrate
php artisan db:seed
```
